# Corporate Risk Radar 2.0

A search engine for corporate risk analysis using financial data and live news sentiment.

## Team Members & Roles
- **Member 1:** Financial Data Lead (data collection & cleaning)
- **Member 2:** News & NLP Lead (news scraping, sentiment analysis)
- **Member 3:** Integrator & Logic Lead (integration, auto-labeling, dashboard logic)

## Project Structure
- `/financial_data` — Financial data scripts and datasets
- `/news_sentiment` — News scraping and sentiment analysis tools
- `/integration_logic` — Core logic for risk scoring and data integration
- `/dashboard` — UI code for the project
- `/docs` — Documentation and screenshots

## Getting Started
1. Clone the repo:  
   `git clone https://github.com/Rohitbarikk/corporate-risk-radar.git`
2. Install requirements (Python, pandas, etc.)
3. Run scripts in each folder as described in the docs.
